# サポート / Support

## 質問・サポートの前に / Before Asking for Support

まず以下をご確認ください。

Please check the following before asking for support.

- [README](../README.md) を読む / Read the README
- 既存の [Issues](../../issues) を検索する / Search existing Issues

---

## サポートの受け方 / How to Get Support

### バグ報告 / Bug Reports

再現可能なバグは **[Issues](../../issues/new/choose)** からご報告ください。

For reproducible bugs, please open an **[Issue](../../issues/new/choose)**.

### 質問・使い方の相談 / Questions & Usage

一般的な質問や使い方の相談は以下からご連絡ください。

For general questions and usage discussions, please reach out via the following.

| 方法 / Method | 連絡先 / Address |
| ------------- | ---------------- |
| メール / Email | [contact@daruks.com](mailto:contact@daruks.com) |
| Discord | [@darui3018823](https://discord.com/users/darui3018823) |

### セキュリティ上の問題 / Security Issues

セキュリティ上の問題は **絶対に** Issue で報告しないでください。  
[SECURITY.md](./SECURITY.md) の手順に従ってご連絡ください。

Please **never** report security issues via Issues.  
Follow the instructions in [SECURITY.md](./SECURITY.md).

---

## 対応範囲について / Scope of Support

このプロジェクトはボランティアベースでメンテナンスされています。  
迅速な対応をお約束することはできませんが、できる限り対応します。

This project is maintained on a volunteer basis.  
We cannot guarantee prompt responses, but we will do our best.
